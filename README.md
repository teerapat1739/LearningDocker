## Install
```
  yarn
```

## Run with node
```
  node server.js
```
  # [http://localhost:3000](http://localhost:3000).
  
## Run with Docker
```
  1. docker build -t sample-nodejs-app .
  2. docker images
  > copy image that you want to run
  3. docker run -p 80:3000 {image-id}
```
  # [localhost](localhost).
